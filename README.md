# AI Code Reviewer

A simple AI-powered Python code reviewer built using Gemini API.

## Features

- Detects syntax errors
- Suggests improvements
- Reviews Python code
- Retry handling for quota issues

## Tech Stack

- Python 3.13
- Gemini API
- google-genai
- dotenv

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ai-code-reviewer.git
cd ai-code-reviewer
